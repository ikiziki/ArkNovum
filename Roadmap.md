# ArkNovum Roadmap

**Building the future of personal hobby preservation, one project at a time.**

This roadmap represents the long-term vision for ArkNovum.

Features and timelines may change as the community grows, priorities shift, and new ideas emerge. The goal is not simply to build software, but to build a platform that can preserve the stories behind the things we create.

---

# Phase 1 — Foundation

**Building the ArkNovum core experience**

Status: 🚧 In Progress

The foundation of ArkNovum begins with a flexible, reliable, and extensible core platform.

Planned features:

- Core application framework
- User authentication and profiles
- Self-hosted deployment
- Database architecture
- Collection management engine
- Media storage system
- Search and discovery
- Tagging system
- Timeline and activity history
- Import and export tools
- Backup and restore capabilities
- Plugin management system

**Goal:**

Create a stable foundation that every hobby plugin can build upon.

---

# Phase 2 — The Plugin Ecosystem

**Transforming ArkNovum into a hobby platform**

Status: 🔨 Development

The power of ArkNovum comes from its modular design.

Planned features:

- Plugin installation system
- Plugin marketplace
- Plugin permissions model
- Community plugin repository
- Plugin documentation framework
- Plugin templates
- Developer tools
- API documentation
- Plugin validation tools

Initial official plugins:

- Gundam / Gunpla Archive
- RC Vehicle Archive
- Model Railroad Archive
- Miniature Painting Archive

**Goal:**

Allow every hobby community to create its own digital archive experience.

# Long-Term Vision

ArkNovum is not just about tracking objects.

It is about preserving:

- The first project
- The difficult project
- The favorite project
- The lessons learned
- The creativity shared
- The memories created

A collection is a record of where you have been.

A project is a record of what you have learned.

ArkNovum exists to preserve both.
